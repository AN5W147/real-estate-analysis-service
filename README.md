# AI 기반 부동산 가격 분석 서비스

> 아파트 가격 정보와 주변 데이터를 분석하여 사용자의 부동산 의사결정을 도와주는 서비스


## 무엇을 하는 프로젝트인가

본 프로젝트는 사용자가 원하는 아파트를 검색하면 
가격 정보, 가격 변화율, 주변 아파트와의 가격 비교 정보를 제공하는 부동산 분석 서비스입니다.

사용자는 여러 정보를 직접 찾아 비교하지 않고 하나의 서비스에서 현재 가격 수준을 확인할 수 있습니다.


## 주요 기능

- 아파트 이름 검색
- 최근 거래 가격 확인
- 가격 변화율 분석
- 주변 아파트 가격 비교
- AI 기반 가격 평가 제공


## 사용 방법

1. 검색창에 아파트 이름 입력
2. 분석하기 버튼 클릭
3. 가격 분석 결과 확인
